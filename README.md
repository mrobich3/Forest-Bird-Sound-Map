# Where the Forest Sings

**An interactive sound map of bird species diversity across Poland Municipal Forest (Poland, Ohio), along a freeway-to-interior gradient.**

Bird species were identified by using [Merlin Bird ID](https://merlin.allaboutbirds.org/) (Cornell Lab of Ornithology) during 4-minute stationary listening sessions at 15 stops, surveyed on six mornings (6‚Äì8 AM) between June 17‚Äì23, 2026. The project tests whether the number of species you can hear rises as you move away from Interstate 680, using distance to the freeway and distance to water as predictors.

**Live map:** https://mrobich3.github.io/Forest-Bird-Sound-Map/

## Reproducing the analysis

**Requirements:** R (4.x) and these packages:

```r
install.packages(c("sf", "osmdata", "leaflet", "dplyr", "tidyr", "readr",
                   "MASS", "stringr", "htmltools", "htmlwidgets",
                   "ggplot2", "ggrepel"))
```

**Set your path.** The script points to a local folder via a single variable at the top:

```r
PROJ <- "C:/Users/mrobi/OneDrive/Documents/Information Design/Final Project/"
```

Change this to the folder where you cloned the repo (keep the trailing slash), e.g. `PROJ <- "C:/Users/you/Forest-Bird-Sound-Map/"`. Every read and write uses this variable, so it's the only path you need to edit.

**Run it** top to bottom. The script will:

1. Load `pins.csv` and `species_long.csv`, lump Black-capped and Carolina Chickadee into "Chickadee Species." (Poland sits in their contact zone).
2. Compute species richness and Northern Cardinal presence per stop, and build the linked bird lists used in the map popups.
3. Query OpenStreetMap for freeway and waterway geometry, compute straight-line distances, and write `pins_with_distances.csv`.
4. Join everything into the master dataset.
5. Build the interactive Leaflet map.
6. Fit the Poisson richness models and the exploratory logistic cardinal model (each with an S08 sensitivity check).
7. Produce the richness-vs-freeway figure (`richness_vs_freeway.png`).

**If the OpenStreetMap step fails:** the distance query (step 3) hits the live Overpass API, which occasionally times out. If it does, the committed `pins_with_distances.csv` already holds the precomputed distances ‚Äî read that file instead of re-running step 3, and the rest of the script works unchanged.

## Data dictionary

### `pins.csv` (15 rows ‚Äî one per stop)
| Column | Meaning |
|---|---|
| `session_id` | Stop identifier, S01‚ÄìS15. |
| `date` | Survey date (M/D/YYYY). |
| `time` | Approximate start time of the listening session. |

| Column | Meaning |
|---|---|
| `session_id` | Stop where the species was detected. |
| `species` | Species name as reported by Merlin. |

### `forest_master_dataset.csv` (15 rows ‚Äî one per stop)
| Column | Meaning |
|---|---|
| `ID` | Row number. |
| `Date`, `Time`, `AM/PM` | Survey date and start time. |
| `Count` | **Species richness** ‚Äî the number of *distinct species* heard at the stop (not a count of individual birds). |

| `Water Distance` | Straight-line meters from the stop to the nearest waterway. |

| `Animals` | Free-text non-bird animals seen (e.g., "2 Squirrels"). |

| `Notes` | Qualitative habitat notes (e.g., "on stream," "swampy," "tall grass field"). |

| `lat`, `lon` | Coordinates (WGS84). |

### `pins_with_distances.csv`
Generated by the script. Adds `dist_water_m` and `dist_freeway_m` (distances in meters, computed from OpenStreetMap geometry) to the stop-level data.

## Notes and caveats

- **Richness, not abundance.** Merlin reports which species are singing, not how many individuals, so the outcome is distinct-species count per stop.
- **Exploratory.** With 15 stops over one week, all models are exploratory. The freeway effect is sensitive to the single freeway-edge stop (S08); the cardinal logistic model is unstable (11 of 15 stops had a cardinal).
- **Distance, not dB, is the noise predictor.** A sound meter at a bird-rich stop is inflated by the very birdsong being counted, so dB is endogenous to the outcome. Distance from the freeway is fixed by geography, so it's a cleaner proxy for traffic noise.
